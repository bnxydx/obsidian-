# 视觉现阶段

## 图像分类与骨干网络（Backbones）

| 模型名称                 | 关键特点                         | 典型应用           | 2025年亮点                     |
| ------------------------ | -------------------------------- | ------------------ | ------------------------------ |
| ConvNeXt v3              | 现代CNN设计，结合Transformer元素 | 通用分类、特征提取 | 高效，ImageNet Top-1 Acc. 88%+ |
| EfficientNet V3          | 复合缩放，轻量级高精度           | 移动端分类         | 边缘设备优化，参数少20%        |
| Vision Transformer (ViT) | Transformer-based，自注意力      | 通用骨干           | 多模态融合，泛化强             |
| Swin Transformer v3      | 分层窗口注意力，高效ViT变种      | 高分辨率分类       | 实时视频分类                   |

## 目标检测（Object Detection）

| 模型名称              | 关键特点                    | 典型应用       | 2025年亮点             |
| --------------------- | --------------------------- | -------------- | ---------------------- |
| YOLOv11 (Ultralytics) | 一阶段实时检测，Anchor-Free | 自动驾驶、安防 | 速度>100 FPS，mAP 55%+ |
| YOLOv12               | 进一步优化多尺度融合        | 无人机检测     | 小目标AP提升10%        |
| RT-DETR               | Transformer-based实时检测   | 工业质检       | 端到端，无NMS          |
| Grounding DINO        | 开放词汇检测，文本引导      | 零样本检测     | 多模态输入，灵活性高   |

## 生成模型与多模态（Generative & Multimodal）

| 模型名称           | 关键特点                 | 典型应用           | 2025年亮点           |
| ------------------ | ------------------------ | ------------------ | -------------------- |
| Stable Diffusion 3 | 扩散模型，文本到图像生成 | 内容创作、数据增强 | 高分辨率，少样本生成 |
| CLIP (OpenAI)      | 对比学习，图像-文本对齐  | 检索、零样本分类   | 多模态基准，工业级   |
| DINOv2             | 自监督ViT，特征学习      | 迁移学习           | 无标签训练，泛化强   |
| Kosmos-2           | 视觉-语言 grounding 模型 | VQA（视觉问答）    | 区域定位，交互式AI   |

## 其他新兴主流模型（Pose Estimation & 3D）

| 模型名称                      | 关键特点           | 典型应用        | 2025年亮点           |
| ----------------------------- | ------------------ | --------------- | -------------------- |
| DenseNet (现代变种)           | 密集连接，参数高效 | 姿态估计        | 轻量级3D重建         |
| NeRF (Neural Radiance Fields) | 神经辐射场，3D重建 | AR/VR、数字孪生 | 实时渲染，移动端支持 |

### 总体趋势总结（2025年）

-   **Transformer主导**：ViT及其变种（如Swin）已成为骨干主流，取代纯CNN。
-   **实时与边缘**：YOLO系列和EfficientNet强调低延迟，适用于IoT/自动驾驶。
-   **多模态融合**：CLIP、SAM 2 等支持文本-视觉交互，推动生成AI和零样本学习。
-   **挑战与方向**：小模型（如MobileViT）和可持续AI（低碳足迹）是热点，预计CVPR 2025将聚焦联邦学习与伦理AI。
