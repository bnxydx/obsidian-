## relu

| 激活函数   | 公式                            | 负半轴行为                      |
| ---------- | ------------------------------- | ------------------------------- |
| ReLU       | f(x) = max(0, x)                | x < 0 时，输出死死为 0          |
| Leaky ReLU | f(x) = max(αx, x)    或分段写： | x < 0 时，输出 αx（有微弱信号） |
|            | f(x) = x       if x > 0         |                                 |
|            | f(x) = αx      if x ≤ 0         |                                 |
