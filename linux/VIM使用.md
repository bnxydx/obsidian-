## 打开文件

~~~
vim filename
~~~

- vim filename: 打开文件，并定位第1行
- vim filename + 一个数字 :打开文件，并定位于第n行
- vim filename +：打开文件，定位至最后一行
- vim filename +/PATTERN : 打开文件，定位至第一次被PATTERN匹配到的行的行首



-----

打开后默认是命令模式
要进入编辑模式

~~~
i 在当前光标的位置  ---- insert
I 在当前行的开头
a 在当前光标位置之后   
A 在当前行的结尾
o 在当前光标下一行
O 在当前光标上一行
~~~

----

## 关闭文件

| 命令模式 | ZZ   | 保存并退出                |
| -------- | ---- | ------------------------- |
| 末行模式 | :q   | 退出。没有动过文件        |
| 末行模式 | :wq  | 保存并退出。动过了,不后悔 |
| 末行模式 | :q!  | 退出。动过了,后悔了       |
| 末行模式 | :w   | 保存，不退出              |



## 进阶

### 上下左右

~~~
k 上
j 下
h 左
l 右
~~~

## 开头结尾

~~~
0 当前行的开头
$ 当前行的结尾
gg 光标到首行
G 光标到结尾
~~~

### 复制粘贴

~~~
yy 复制当前行
nyy n是数字，复制当前行和下面的n行
p 粘贴
~~~

### 撤销修改

~~~
u 
ctrl + r 反向撤销修改
~~~

## 删除

~~~
dd 删除当前行
ndd n是数字，表示删除当前光标向下n行
dG 从当前行开始，向下删除全部
dgg 从当前行开始，向上删除全部
d0 从当前光标开始，删除到本行的开头
~~~

## 设置行数

~~~
末行模式   :set number
		   或者set nu
		   
永久设置行数
cd ~
一般是没有 .vimrc 这个文件夹的
vim .vimrc 创建这个文件
写 set nu
. .vimrc 执行这个文件

~~~
