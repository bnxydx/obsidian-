# 字符的判断

## c++

~~~
isalnum() — 检查一个字符是否是字母或数字。如果是，返回非零值；如果不是，返回0。
isdigit() — 检查一个字符是否是数字（'0'-'9'）。如果是，返回非零值；如果不是，返回0。
isalpha() — 检查一个字符是否是字母（'a'-'z' 或 'A'-'Z'）。如果是，返回非零值；如果不是，返回0。
~~~

## python

~~~
在Python中，isalnum(), isdigit(), 和 isalpha() 方法是直接应用于字符串对象上的。这些方法不需要导入额外的库，因为它们是Python字符串类的一部分
~~~
