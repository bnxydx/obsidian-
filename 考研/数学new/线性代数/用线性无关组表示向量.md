在处理这类矩阵问题时，关键在于理解行最简形矩阵（Reduced Row Echelon Form, RREF）与向量线性组合之间的关系。

图中的 $\gamma_1, \gamma_2, \dots, \gamma_5$ 代表行最简矩阵的列向量，而 $\alpha_1, \alpha_2, \dots, \alpha_5$ 代表原始矩阵的列向量。**核心原理是：初等行变换不改变列向量之间的线性关系。**

以下是推导 $\gamma_3$ 和 $\gamma_5$（对应你问的 $x_3$ 和 $x_5$ 列）的具体步骤：

### 1. 确定基向量（主元列）

首先圈出每一行的第一个 $1$（主元）：

-   第 1 列 $\gamma_1 = \begin{pmatrix} 1 \\ 0 \\ 0 \\ 0 \end{pmatrix}$
-   第 2 列 $\gamma_2 = \begin{pmatrix} 0 \\ 1 \\ 0 \\ 0 \end{pmatrix}$
-   第 4 列 $\gamma_4 = \begin{pmatrix} 0 \\ 0 \\ 1 \\ 0 \end{pmatrix}$

这三列是单位坐标向量，构成了该向量空间的基。

------

### 2. 计算 $\gamma_3$ 的线性组合

观察矩阵的第 3 列：

$$\gamma_3 = \begin{pmatrix} -1 \\ -1 \\ 0 \\ 0 \end{pmatrix}$$

我们可以直接利用基向量将其分解：

$$\gamma_3 = (-1) \cdot \begin{pmatrix} 1 \\ 0 \\ 0 \\ 0 \end{pmatrix} + (-1) \cdot \begin{pmatrix} 0 \\ 1 \\ 0 \\ 0 \end{pmatrix} + 0 \cdot \begin{pmatrix} 0 \\ 0 \\ 1 \\ 0 \end{pmatrix}$$

即：**$\gamma_3 = -\gamma_1 - \gamma_2$**

由于线性关系在变换前后保持不变，所以对应的原矩阵向量也有：**$\alpha_3 = -\alpha_1 - \alpha_2$**。

------

### 3. 计算 $\gamma_5$ 的线性组合

观察矩阵的第 5 列：

$$\gamma_5 = \begin{pmatrix} 4 \\ 3 \\ -3 \\ 0 \end{pmatrix}$$

同理，利用对应的基向量分量直接读出系数：

-   第 1 行的数字是 $4$，对应 $\gamma_1$
-   第 2 行的数字是 $3$，对应 $\gamma_2$
-   第 3 行的数字是 $-3$，对应 $\gamma_4$（注意：这里对应的是第 4 列的主元）

所以：**$\gamma_5 = 4\gamma_1 + 3\gamma_2 - 3\gamma_4$**

对应原矩阵向量：**$\alpha_5 = 4\alpha_1 + 3\alpha_2 - 3\alpha_4$**。

------

### 总结技巧

在**行最简形**中求某一列（非主元列）如何由其他主元列表示，其实非常简单：

>   **直接看该列的数字。** 该列在第几行有数字，就对应第几个主元列，系数就是该位置的数值。

-   $\gamma_3$ 的第 1 行是 $-1$，第 2 行是 $-1 \rightarrow$ 系数就是 $-1$ 和 $-1$。
-   $\gamma_5$ 的第 1、2、3 行分别是 $4, 3, -3 \rightarrow$ 对应前三个主元列（即第 1, 2, 4 列）的系数。
