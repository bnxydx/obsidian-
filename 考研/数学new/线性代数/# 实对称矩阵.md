# 正交矩阵

$AA^T = A^TA = E$

## 性质

1.   $A^{-1} = A^T$
2.   |A| = 1 或-1
3.   行/列向量长度为1，并且两两正交

~~~
1 0 0 
0 1 0
0 0 1
~~~
