若AB = 0则r(A) + r(B)  <= n

要证r(A) + r(B)  <= n

即证 r(B) <= n - r(A)

把B列展开

A(b1.b2,...bs) = 0

即Ab_i = 0是Ax = 0的解

B的每一列都是Ax = 0的解





------

### 第一步：引入“解空间”的概念

既然 $B$ 的每一列（即 $b_1, b_2, \dots, b_s$）都是齐次线性方程组 $Ax = 0$ 的解，那么 $B$ 的所有列向量组成的集合，一定是 $Ax = 0$ **基础解系**所能表示的空间（即解空间）的一个**子集**。

由此可知：

**$B$ 的列向量相关性所能撑起的“维度”，绝对不会超过 $Ax = 0$ 的解空间的“总维度”。**

用数学语言表达就是：

$$r(B) \le \text{方程组 } Ax = 0 \text{ 的解空间的维数}$$

------

### 第二步：解空间的维数是多少？

这是线性代数中的一个核心定理：

对于一个 $n$ 元齐次线性方程组 $Ax = 0$，如果矩阵 $A$ 的秩为 $r(A)$，那么它的基础解系含有的向量个数（即解空间的维数）为：

$$\text{维数} = n - r(A)$$

------

### 第三步：合并完成证明

把第一步的不等式和第二步的等式合并在一起：

$$r(B) \le n - r(A)$$

最后，把 $-r(A)$ 移项到不等式的左边，就得到了你要的结论：

$$r(A) + r(B) \le n$$

------

### 💡 为什么这个结论很重要？

在 408 考研或者数据结构的高级算法分析中，这个性质经常用来判断矩阵的线性相关性：

-   如果 $AB = 0$，说明 $B$ 里的信息“太少了”或者“太整齐了”，以至于被 $A$ 一下子全部过滤掉了。
-   如果 $A$ 是满秩的（$r(A) = n$），那么 $r(B)$ 必须为 0，即 $B$ 只能是零矩阵。
