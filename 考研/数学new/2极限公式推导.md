既然想从**其他已知不等式代换**的角度来推导，那我们其实可以从你提到的 $\ln x < x$ 的变体出发。

这里有两个非常漂亮的代换路径，你可以看看哪一种更符合你当时的直觉：

### 途径一：从最经典的 $\ln t \le t - 1$ 进行变量代换

你直觉里想到的 $\ln x < x$（更准确地说是 $\ln t \le t - 1$，在 $t=1$ 时取等号）确实是通往这个公式的钥匙，只不过不能直接套，而是需要做一次**倒数代换**。

1. 我们熟知的基础不等式（对所有 $t > 0$ 成立）：
    
    $$\ln t \le t - 1$$
    
2. 关键的一步来了：我们把里面的 $t$ 替换成 $\frac{1}{1+x}$（其中 $x > 0$）。代入后得到：
    
    $$\ln\left(\frac{1}{1+x}\right) \le \frac{1}{1+x} - 1$$
    
3. 接下来利用对数的性质展开左边，并对右边进行通分：
    
    - 左边：$\ln\left(\frac{1}{1+x}\right) = -\ln(1+x)$
        
    - 右边：$\frac{1}{1+x} - \frac{1+x}{1+x} = \frac{1 - (1+x)}{1+x} = -\frac{x}{1+x}$
        
4. 把这两步拼回去：
    
    $$-\ln(1+x) \le -\frac{x}{1+x}$$
    
5. 两边同乘以 $-1$，**不等号方向取反**：
    
    $$\ln(1+x) \ge \frac{x}{1+x}$$
    

因为 $x > 0$ 时取不到等号，所以 $\frac{x}{1+x} < \ln(1+x)$。这个方法绕了一圈倒数和负号，正好能把你直觉里的那个基础不等式给套用上！

### 途径二：从积分不等式直接代换（几何直观）

如果你学过定积分，这个式子可以用**对数函数的积分定义**极其自然地代换出来。

1. 首先，我们知道当 $t > 0$ 时，有一个显然的式子：
    
    $$\frac{1}{1+x} \le \frac{1}{t}$$
    
    （这个式子在 $t$ 从 $1$ 变到 $1+x$ 的区间内是显然成立的，因为分母越大，分式越小）。
    
2. 我们对这个不等式在区间 $[1, 1+x]$ 上同时给它套上积分 $\int_{1}^{1+x} \text{d}t$：
    
    $$\int_{1}^{1+x} \frac{1}{1+x} \text{d}t < \int_{1}^{1+x} \frac{1}{t} \text{d}t$$
    
3. 此时两边分别积出来：
    
    - 左边：因为 $\frac{1}{1+x}$ 相对于自变量 $t$ 是个常数，直接提出来，积分结果就是 `常数 × 区间长度`：
        
        $$\frac{1}{1+x} \cdot ((1+x) - 1) = \frac{x}{1+x}$$
        
    - 右边：$\frac{1}{t}$ 的原函数就是 $\ln t$：
        
        $$\ln t \Big|_{1}^{1+x} = \ln(1+x) - \ln 1 = \ln(1+x)$$
        
4. 两边一组合，立马得到：
    
    $$\frac{x}{1+x} < \ln(1+x)$$
    

这个积分代换在高等数学的证明题里非常高级，它在几何上其实就是用一个**矩形的面积**去压了 $\frac{1}{t}$ 曲线下的**曲边梯形面积**。你看看这两种代换思路，有没有对上你当时的灵感？

# $(1+x)^\alpha$

$$(1 + t)^\alpha = 1 + \alpha t + \frac{\alpha(\alpha - 1)}{2}t^2 + o(t^2)$$

这个方法的核心思想是：**化未知为已知**。我们把复杂的指数幂 $(1+t)^\alpha$ 通过自然对数 $e^{\ln(\dots)}$ 转换成我们最熟悉的两个标准等价无穷小：

1. 当 $x \to 0$ 时，$e^x - 1 \sim x \implies e^x = 1 + x + o(x)$
    
2. 当 $x \to 0$ 时，$\ln(1+t) \sim t \implies \ln(1+t) = t - \frac{1}{2}t^2 + o(t^2)$
    

我们要推导下一项（二次项），只需要把这两个基础代换稍微写精确一点。

### 换元推导三步走

#### 第一步：用 $e$ 和 $\ln$ 强行改写

任何正数的幂都可以写成以 $e$ 为底的形式：

$$(1+t)^\alpha = e^{\ln(1+t)^\alpha} = e^{\alpha \ln(1+t)}$$

#### 第二步：代换内部的 $\ln(1+t)$

因为我们要找二次项，所以把 $\ln(1+t)$ 展开到 $t^2$ 项：

$$\ln(1+t) = t - \frac{1}{2}t^2 + o(t^2)$$

整体乘以 $\alpha$，得到指数位置的精确表达式：

$$\alpha \ln(1+t) = \alpha t - \frac{\alpha}{2}t^2 + o(t^2)$$

#### 第三步：代换外部的 $e^x$

现在，我们把整个 $\left( \alpha t - \frac{\alpha}{2}t^2 \right)$ 看成一个整体 $x$。当 $t \to 0$ 时，这个整体 $x$ 显然也趋于 0。

利用 $e^x$ 的标准展开式 $e^x = 1 + x + \frac{1}{2}x^2 + o(x^2)$，把整体代入进去（记住，我们只需要保留到 $t^2$ 的项）：

$$e^{\left( \alpha t - \frac{\alpha}{2}t^2 \right)} = 1 + \underbrace{\left( \alpha t - \frac{\alpha}{2}t^2 \right)}_{\text{对应 } x} + \frac{1}{2}\underbrace{\left( \alpha t - \frac{\alpha}{2}t^2 \right)^2}_{\text{对应 } x^2} + o(t^2)$$

对上面这个式子进行化简，只留下不大于 2 次方的部分：

1. **一次项和第一部分的二次项**：$\alpha t - \frac{\alpha}{2}t^2$
    
2. **第二部分的平方项**：$\frac{1}{2}\left( \alpha t - \frac{\alpha}{2}t^2 \right)^2 \approx \frac{1}{2}(\alpha t)^2 = \frac{\alpha^2}{2}t^2$ （更高级的项由于超过了 2 次方，直接扔进 $o(t^2)$）
    

#### 第四步：合并同类项

把所有的 $t^2$ 项打包合并：

$$(1+t)^\alpha = 1 + \alpha t - \frac{\alpha}{2}t^2 + \frac{\alpha^2}{2}t^2 + o(t^2)$$

$$(1+t)^\alpha = 1 + \alpha t + \left( \frac{\alpha^2 - \alpha}{2} \right)t^2 + o(t^2)$$

提取公因式 $\alpha$，分子就变成了 $\alpha(\alpha-1)$：

$$(1+t)^\alpha = 1 + \alpha t + \frac{\alpha(\alpha-1)}{2}t^2 + o(t^2)$$

### 总结

这个方法之所以简单，是因为它把**求导**的功夫，变成了**多项式乘法和提取公因式**。在考研或者极限化简中，如果遇到复杂的复合函数，这种“由内而外”层层代换的思路是最不容易出错的！
