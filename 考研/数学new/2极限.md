（一）极限的概念

# 数列极限: $\lim_{n\to\infty} a_n = A$:

$\forall \varepsilon > 0, \exists N > 0$, 当 $n > N$ 时, $|a_n - A| < \varepsilon$.

【注】
（1）几何意义;
	$$
A-\varepsilon <a_n\ <A+\varepsilon 
$$
前N项无所谓（有限项必有界），但是之后的无限项需要满足
=> 
若 $a < A < b$，则 $\exists N$，当 $n > N$ 时，$a < a_n < b$。

（2）数列 $\{x_n\}$ 的极限与前有限项无关;只和后面无限项有关

（3）奇偶$\lim_{n \to \infty} x_n = a \iff \lim_{k \to \infty} x_{2k-1} = \lim_{k \to \infty} x_{2k} = a$.

如果基数列的极限和偶数列的极限都**存在且相等**才能推出原数列极限存在且等于A

>如果没有相等条件:反例：xn = $(-1)^n$

(4)如果一个数列有最大值，那么一定存在一项大于等于极限值
反过来
如果有一项大于等于极限值，那么数列有最大值
即：**一个数列有最大值的充要条件是一个存在一项大于等于极限值**
若 $\lim_{n \to \infty} a_n = a$，则 $\{a_n\}$ 有最大值 <= =>存在 $a_n \geq a$。

同样的：
**一个数列有最小值的充要条件是一个存在一项小于等于极限值**
若 $\lim_{n \to \infty} a_n = a$，则 $\{a_n\}$ 有最小值 <= =>存在 $a_n <= a$。

# 函数极限

1) $\lim\limits_{x \to +\infty} f(x) = A$: $\forall \varepsilon > 0, \exists X > 0$，当 $x > X$ 时，$|f(x) - A| < \varepsilon$.

2) $\lim\limits_{x \to -\infty} f(x) = A$: $\forall \varepsilon > 0, \exists X > 0$，当 $x < -X$ 时，$|f(x) - A| < \varepsilon$.

3) $\lim\limits_{x \to \infty} f(x) = A$: $\forall \varepsilon > 0, \exists X > 0$，当 $|x| > X$ 时，$|f(x) - A| < \varepsilon$.
