# YOLO死鸡检测系统 - 页面使用说明

## 🎉 页面已创建完成

### 页面列表

1. **首页** - `/yolo/`
2. **图片检测** - `/yolo/upload/`
3. **检测记录** - `/yolo/list/`
4. **房间管理** - `/yolo/rooms-page/`

## 📱 页面功能说明

### 1. 首页 (yolo_index.html)

**访问地址**: `http://localhost:8000/yolo/`

**功能**:
- 系统入口页面
- 提供三个功能入口卡片
- 精美的渐变背景和动画效果

**特点**:
- 响应式设计，支持移动端
- 卡片悬停动画效果
- 清晰的导航结构

---

### 2. 图片检测页面 (yolo_upload.html)

**访问地址**: `http://localhost:8000/yolo/upload/`

**功能**:

#### 单张检测模式
- 选择鸡舍（必填）
- 设置置信度阈值（可选，默认0.25）
- 上传单张图片
- 支持拖拽上传
- 实时预览
- 显示检测结果（原图 vs 检测结果）

#### 批量检测模式
- 选择鸡舍（必填）
- 设置置信度阈值（可选，默认0.25）
- 一次上传多张图片
- 支持拖拽上传
- 批量预览
- 显示每张图片的检测结果统计

**API接口**:
- 单张检测: `POST /yolo/api/detect/`
- 批量检测: `POST /yolo/api/batch-detect/`

**返回信息**:
- 检测置信度
- 检测对象数量
- 原图URL
- 检测结果图URL
- 数据库记录ID

---

### 3. 检测记录页面 (yolo_detect_list.html)

**访问地址**: `http://localhost:8000/yolo/list/`

**功能**:

#### 筛选功能
- 按鸡舍名称筛选
- 按日期筛选（查询当天所有记录）
- 支持组合筛选
- 重置筛选

#### 显示功能
- 表格展示所有检测记录
- 显示: ID、鸡舍、检测图片、检测时间
- 图片缩略图预览
- 点击图片查看大图（模态框）
- 分页功能（每页10条）

#### 统计信息
- 总记录数
- 当前页码
- 每页显示数量

**API接口**:
- 查询记录: `GET /yolo/api/detect-list/`
- 支持参数: `page`, `room_name`, `date`

---

### 4. 房间管理页面 (yolo_room_manage.html)

**访问地址**: `http://localhost:8000/yolo/rooms-page/`

**功能**:

#### 添加鸡舍
- 输入鸡舍名称
- 实时验证（名称唯一性）
- 添加成功提示

#### 鸡舍列表
- 卡片式展示所有鸡舍
- 显示: 鸡舍名称、创建时间、更新时间、ID
- 卡片悬停动画效果

#### 统计信息
- 鸡舍总数

**API接口**:
- 获取列表: `GET /yolo/api/rooms/`
- 添加鸡舍: `POST /yolo/api/rooms/`

## 🚀 快速开始

### 1. 启动服务

```bash
# 进入项目目录
cd c:\Users\Lenovo\Desktop\django-demo\demo

# 激活虚拟环境
..\demo-env\Scripts\activate

# 运行服务器
python manage.py runserver
```

### 2. 访问系统

打开浏览器访问: `http://localhost:8000/yolo/`

### 3. 使用流程

#### 首次使用:
1. 访问首页 → 进入房间管理
2. 添加鸡舍（例如：1号鸡舍、2号鸡舍）
3. 返回首页 → 进入图片检测
4. 选择鸡舍，上传图片进行检测
5. 查看检测记录

#### 日常使用:
1. 首页 → 图片检测
2. 选择鸡舍，上传图片
3. 查看检测结果
4. 检测记录中查询历史数据

## 🔌 API接口总结

### 检测相关

```bash
# 单张检测
POST /yolo/api/detect/
参数: file(图片), room_name(鸡舍名), conf(置信度)

# 批量检测
POST /yolo/api/batch-detect/
参数: files(多个图片), room_name(鸡舍名), conf(置信度)

# 查询记录
GET /yolo/api/detect-list/?room_name=1号鸡舍&date=2025-10-22&page=1
```

### 房间管理

```bash
# 获取房间列表
GET /yolo/api/rooms/

# 添加房间
POST /yolo/api/rooms/
参数: {"room_name": "1号鸡舍"}
```

### 其他

```bash
# 发送邮件通知
POST /yolo/api/sendemail/
参数: score(分数), Qnum(QQ号)
```

## 💡 使用技巧

### 1. 批量检测
- 一次可以选择多张图片
- 按住 Ctrl 键选择多个文件
- 或者直接拖拽多个文件到上传区域

### 2. 查询记录
- 可以只选择鸡舍查询
- 可以只选择日期查询
- 可以同时选择鸡舍和日期精确查询

### 3. 查看大图
- 点击表格中的缩略图查看大图
- 点击模态框外部关闭大图

### 4. 置信度设置
- 默认0.25，范围0-1
- 值越大，检测越严格
- 建议根据实际情况调整

## 🐛 常见问题

### 1. 无法上传图片
**解决**: 检查文件格式是否为 JPG/PNG/BMP/GIF

### 2. 检测失败
**解决**: 
- 确保已选择鸡舍
- 检查网络连接
- 查看浏览器控制台错误信息

### 3. 查不到记录
**解决**:
- 确认筛选条件是否正确
- 尝试重置筛选
- 确认该日期是否有检测记录

### 4. 添加鸡舍失败
**解决**:
- 检查鸡舍名称是否重复
- 鸡舍名称不能为空
- 名称长度不超过50字符
